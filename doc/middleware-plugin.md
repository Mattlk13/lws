## middleware-plugin
MiddlewareStack plugin API.

**Example**  
```js
class Greeter {
  middleware (config) {
    return async (ctx, next) => {
      ctx.response.body = `Hello Mr ${config.surname}`
      await next()
    }
  }

  optionDefinitions () {
    return [
      { name: 'surname', description: 'Your family name.' }
    ]
  }
}

export default Greeter
```
